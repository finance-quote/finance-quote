Welcome to the Finance::Quote webpage.

Finance::Quote is a perl module which can be used to obtain information from a variety of sources, including markets in Australia, USA, Canada, Europe, and a number of managed funds.

The latest version of Finance::Quote is always available via CPAN, or our download page.

News

2000-Sep-16Version 1.02 of Finance::Quote is out. The new version contains many bugfixes, including much better handling of Australian markets. Also included are new modules to fetch information regarding UK Unit Funds, Deutsche Bank Gruppe funds, and stocks from Vereinigte Wirtschaftsdienste GmbH.

2000-Jul-25Version 1.00 of Finance::Quote has been released. New features include currency lookups, automatic currency conversion, failover lists, better error handling, and loadable module support. The new release can be downloaded here.
